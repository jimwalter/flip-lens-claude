Flip Scout Settings

Anthropic API key

Optional. When set, Flip Scout sends your cropped screenshot to a Claude vision model to generate a short search keyword, which is used to pre-fill eBay sold-listings and Google Shopping searches. Without a key, capture, clipboard copy, and Google Images still work exactly the same — the eBay/Shopping steps are simply skipped.

Your key is stored only on this device (chrome.storage.local, never synced) and is sent directly to Anthropic's API — never to any other server. Get a key at console.anthropic.com.

Keyboard shortcut

Default capture shortcut is Ctrl+Shift+Y (Windows/Linux) or Cmd+Shift+Y (Mac). Chrome silently skips binding it if something else on your system already uses it — check and remap it at chrome://extensions/shortcuts.
